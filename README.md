

# 토이프로젝트02 OTT 프로젝트

OTT 토이프로젝트 입니다.

OTT 토이프로젝트는

Vue를 메인으로 사용하며
Vuex를 사용하여 State 를 관리하고 있으며 새로고침할 시 데이터가 삭제됩니다.

실제로 영상을 관리하지 않으며 Yotube API를 사용하여 OTT 와 비슷하게 만든 프로젝트 입니다.
